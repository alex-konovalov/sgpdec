News

SgpDec is switching to GAP 4.5. The last SgpDec release for GAP 4.4 is sgpdec-0.5.38. Therefore the development version (svn repository) works only with the upcoming release (the beta release can be downloaded from here). Also, a new package dependency is introduced Citrus, replacing Monoid. These changes make the installation of SgpDec slightly more challenging, but once GAP 4.5 is out, everything will be back to normal.
